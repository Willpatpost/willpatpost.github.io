Selected Projects

Python NumPy Neural Networks Browser Demo
Live Demo Source Code

A handwritten digit recognition application built with Python, NumPy, and Tkinter, with the neural network, optimizer, regularization, model persistence, and preprocessing pipeline implemented from scratch. The deployed browser demo uses an exported copy of the trained network to run preprocessing and inference locally in JavaScript.

  • Two-hidden-layer neural network with batch normalization, ReLU, dropout, Adam, and softmax classification
  • Interactive drawing canvas with cropping, scaling, centering, ranked predictions, and confidence scores
  • Static GitHub Pages demo that performs client-side inference without a backend
Python BFS / DFS Greedy Search A*
Live Demo Source Code

Sokomind is a Sokoban variant with generic and letter-matched boxes, playable as a Python desktop application or a static browser game. The solver models puzzle states and compares DFS, BFS, Greedy, and A* search strategies across increasingly complex levels.

  • State-space search over robot movement, legal pushes, walls, storage goals, and box constraints
  • Multiple solving strategies for comparing completeness, speed, and heuristic behavior
  • Static GitHub Pages deployment with a playable web interface
Constraint Satisfaction Backtracking Lexicon Search
Live Demo Source Code

A constraint satisfaction system for crossword generation and solving, built on backtracking search over the ENWiktionary lexicon (~10M entries). Word placement is modeled as variable assignment with intersection constraints at shared grid cells; the solver prunes invalid partial assignments to reduce search depth and supports end-to-end puzzle authoring and deployment.

Java Algorithms Interactive UI
Source Code

An N-puzzle engine with configurable grid dimensions (3×3-7×7). Game state is encoded as a tile permutation with a blank sentinel; solvability is verified via inversion-count parity before shuffle. Moves apply adjacent tile swaps relative to the blank index with O(1) state updates, plus runtime telemetry for elapsed time and move count.

Python NumPy Pillow Evolutionary Computing
Source Code

An evolutionary computation pipeline that reconstructs grayscale target images from populations of semi-transparent triangular primitives. Each chromosome encodes vertex coordinates, luminance, and alpha-blending parameters; fitness is computed as normalized mean squared error against a Gaussian-blurred reference image.

  • Raster composition via alpha-weighted triangle overlays on a canvas buffer
  • Tunable GA hyperparameters: population size, mutation rate, crossover rate, and generation limits
  • Parallel fitness evaluation using multithreaded batch scoring across individuals
  • Preprocessing pipeline with Gaussian blur to stabilize pixel-level error gradients
  • Serialized frame output compiled into an evolution GIF for convergence visualization
